Sistema de Gestão Estudantil
Esse aplicativo é uma ferramenta para os administradores gerenciarem as informações dos alunos, professores, disciplinas e matrículas.


- Pré-requisitos
Antes de poder usar esse programa, você precisa ter o seguintes softwares instalado:
Python 3
Sqlite3 
Flask 

Instalando

Clone o repositório para sua máquina local.
Crie um ambiente virtual usando python -m venv env.
Ative o ambiente virtual usando source env/bin/activate.
Instale as dependências usando pip install -r requirements.txt.

Uso 

Autentique-se com o login e a senha
Escolha a aplicação: Modo Server ou Sistema de Gerenciamento Estudantil
